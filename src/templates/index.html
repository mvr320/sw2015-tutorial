Welcome to the SW2015 Sandwich Generator Server

About the Program

This program is meant to demonstrate an online sandwich generator by using SPARQL queries. My ontology focussed mainly on new sandwich ideas where for example new kinds of meat also give an opportunity for a (maybe) new amazing sandwich.

Using the Program

The menu is an example of the datausage with and without reasoning. The first menu usages reasoning and gets results for all inferences. SaltedPork which is equivalent to Ham is considered Ham and has all the properties of this. The second menu doesn't have this inference which means that all the subclasses and individuals of SaltedPork aren't considered Ham. For a good overview one should look at all the individuals and subclasses of Thing (the toplevel class in the ontology, between the two menus there is a big differences in the amount of submembers.
